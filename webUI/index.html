Lookup

Publish
`write_outbox` is the automatic local dump mode. When enabled, eligible verified matches are written to the outbox as downloads complete. The publisher worker reads that outbox on a schedule, writes dataset updates to an auto-managed `retreivr-community-publish/` branch, and can open or update a pull request automatically. GitHub auth is read from the configured environment variable, not from the UI.
